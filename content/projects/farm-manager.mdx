## Overview
Farm Manager is a comprehensive web application designed to help farm operators manage their agricultural operations efficiently. The application provides tools for tracking farm activities, managing finances, monitoring production cycles, and analyzing performance metrics.

## Features
- **Farm Management**: Create and manage multiple farms with detailed information
- **Production Cycle Tracking**: Monitor production cycles with start/end dates and status
- **Financial Management**: Track expenses, sales, and profitability
- **Inventory Management**: Manage assets, feeds, and other resources
- **Client Management**: Maintain customer and supplier information
- **Reporting**: Generate reports on various aspects of farm operations
- **Multi-currency Support**: Work with different currencies for financial transactions
- **Data Export/Import**: Import and export data in various formats (CSV, Excel)
- **AI-Powered Assistant**: Natural language interface for managing farm operations through the Flame Assistant

## Flame Assistant
The Flame Assistant is an AI-powered agent designed to help manage farm operations through natural language interactions. It integrates with the Farm Manager application to provide a conversational interface for managing farms, cycles, expenses, sales, and clients.

### Features
- Natural language interface for farm management
- Real-time data access and updates
- Support for multiple farms and production cycles
- Expense and sales tracking
- Client management
- Multi-currency support
