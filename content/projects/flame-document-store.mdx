## Overview
A modern, AI-powered document management and retrieval system built with Next.js and FastAPI. The system uses ColFlor (Contextual Late Interaction over BERT for Efficient Retrieval) for advanced visual document understanding and semantic search capabilities.

### Document Management
- **Multi-format Support**: Upload and process images (PNG, JPG, JPEG, TIFF, BMP, GIF, WebP) and PDF documents
- **Batch Processing**: Process multiple PDF pages automatically in batches of 4
- **Document Grouping**: Automatically groups PDF pages from the same document
- **Preview System**: Real-time preview of documents before processing

### AI-Powered Search
- **Visual Document Understanding**: Uses ColFlor model for advanced image-text retrieval
- **Semantic Search**: Find documents based on content meaning, not just keywords
- **Multi-Collection Search**: Search across multiple collections simultaneously
- **Score-based Results**: Results ranked by relevance scores

### Organization System
- **Stores**: Top-level containers for organizing document collections
- **Collections**: Logical groupings of related documents within stores
- **Metadata Management**: Rich metadata storage for documents and collections
